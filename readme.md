# taplet

Create HTML from TAP reports.

[![NPM Badge](https://nodei.co/npm/taplet.png)](https://npmjs.com/package/taplet)

## Install

```sh
npm install --global taplet
```

## Usage

```
$ tape test.js | taplet
<!DOCTYPE html>...

$ tape test.js | taplet --outputFile report.html
```
